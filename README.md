## Como publicar (passo a passo)

### 1. Criar conta no GitHub
- Vai a https://github.com e cria uma conta gratuita

### 2. Criar repositório
- Clica em "New repository"
- Nome: `alaine-app`
- Deixa público
- Clica "Create repository"

### 3. Fazer upload dos ficheiros
- Clica "uploading an existing file"
- Arrasta TODOS os ficheiros desta pasta para lá
- Escreve "Alaine app" na mensagem e clica "Commit changes"

### 4. Criar conta no Vercel
- Vai a https://vercel.com e cria conta (pode ser com o GitHub)

### 5. Ligar o projeto
- Clica "Add New Project"
- Seleciona o repositório `alaine-app`
- Clica "Deploy"

### 6. Adicionar a chave da API Anthropic
- No painel do Vercel, vai a Settings → Environment Variables
- Adiciona:
  - Nome: `ANTHROPIC_API_KEY`
  - Valor: a tua chave da Anthropic (começa com `sk-ant-...`)
- Clica Save e depois faz Redeploy

### 7. Instalar no telemóvel
- Abre o link do Vercel no Chrome (Android) ou Safari (iPhone)
- No Chrome: Menu (⋮) → "Adicionar ao ecrã principal"
- No Safari: Partilhar → "Adicionar ao ecrã de início"
- A app aparece no teu ecrã como qualquer outra! ⭐

---

## Obter a chave da API Anthropic
- Vai a https://console.anthropic.com
- Cria conta e vai a "API Keys"
- Clica "Create Key" e copia o valor
